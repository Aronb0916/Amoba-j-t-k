# Amoba-j-t-k
Készítette: Bogár Áron (JMSN9I)
Projektmunka: Egy amőba játék készítése ahol egy gépi ellenféllel kell játszani,
5 szimbólum akkor nyer ha vertikálisan, diagonálisan vagy horizontálisan helyezkedik el.
Bekér egy játékosnevet és elmenti JSON fájlba azt az adott játékmenetet amiből a felhasználó kilépett.
